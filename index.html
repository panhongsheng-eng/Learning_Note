Linear Programming And Simplex Algorithm

 

👨‍🏫1.What is Linear Programming?

一般来说，所有具有线性目标和线性等式/不等式约束的问题都可以被视为线性规划。然而，有一些被广泛接受的公式。

(LP.Basic)minc⊤xs.t.Ax≤b

其中：

  • x∈Rn：决策变量向量。

  • c∈Rn：目标函数系数。

  • A∈Rm×n，b∈Rm：约束矩阵与向量。

  • 约束 Ax≤b 是逐元素解释的：ai⊤x≤bi。

标准形式。这种形式在可视化方面似乎是最直观和几何化的。设有向量 c∈Rn、b∈Rm，以及矩阵 A∈Rm×n。

(LP.Standard)minx∈Rnc⊤xs.t.Ax=bxi≥0,i=1,…,n

Ax≤b⟺{Ax+z=bz≥0

🔢将无符号变量转化为非负变量：

x⟺{x=x+−x−x+≥0x−≥0

其中，z 是松弛变量，用来将不等式约束转为等式约束；而 x+ 和 x− 是将任意实数变量拆分成两个非负变量的表示方法。

这个问题等价于以下线性规划形式，通过引入向量变量 t∈Rn 替代坐标绝对值：

{mint∈Rn,x∈Rn1⊤ts.t. ai⊤x−bi≤ti,i=1,…,n−ai⊤x+bi≤ti,i=1,…,n

以上将这两个非线性目标函数转化为线性规划问题，方便用线性规划方法求解。

4 Duality

对偶问题的四种可能情况：

在线性规划中，原问题（Primal）与对偶问题（Dual）之间存在以下四种情况：

  1. 💡原问题与对偶问题均无可行解（不可行）。

  2. 💡原问题不可行，但对偶问题无界（即最优值为 −∞ 或 +∞）。

  3. 💡原问题无界，但对偶问题不可行。

  4. 💡原问题和对偶问题均有可行解，且它们的最优值相等（👨‍🏫这就是强对偶性👨‍🏫）。

示例：标准形式线性规划及其对偶问题

给定如下标准形式的线性规划：

{minx∈Rnc⊤xs.t. Ax=bxi≥0,i=1,…,n

其对偶问题为：

{maxy∈Rmb⊤ys.t. A⊤y⪯c
任务

题目要求你找出上述原问题的对偶问题，并确认对偶问题的表达式是否与上述形式一致。

补充说明

💡这里，x 是原问题的决策变量，y 是对偶问题的变量（拉格朗日乘子）。

💡约束 A⊤y⪯c 表示对偶变量 y 满足对应的线性不等式。

💡对偶问题最大化 b⊤y，而原问题最小化 c⊤x。

💡强对偶性保证两者最优值相等（当两者均可行时）。

5 Idea of simplex algorithm

单纯形算法沿着多面体的边缘行走，在每个顶点处选择能够使目标函数 c⊤x 下降最多的边。该过程要么在某个顶点处终止，要么沿着无约束的边继续前进（导致目标值趋于负无穷）。

我们将在简单的不等式形式线性规划中说明单纯形算法：

(LP.Inequality){minx∈Rnc⊤xs.t. Ax≤b

定义：基 B 是从 1 到 m 的整数中选出的一个大小为 n 的子集，使得子矩阵 AB 的秩为 n。注意，我们可以将子矩阵 AB 及对应的右侧向量 bB 与基 B 关联。同时，可以由该基导出这些超平面的交点：xB=AB−1bB。若 AxB≤b，则基 B 是可行的。如果 xB 是该不等式形式线性规划的最优解，则基 B 是最优的

💡定理

✔️如果标准形式线性规划的可行域非空，则至少存在一个基本可行点。 ✔️如果标准形式线性规划有解，则至少存在一个基本最优解。 ✔️如果标准形式线性规划是可行且有界的，那么它必定存在最优解

由于我们有一个基 B，可以在该基下将目标向量 c 分解，并找到标量系数 λB：
λB⊤AB=c⊤⟺λB⊤=c⊤AB−1
定理：如果 λB 的所有分量都不为正（即 λB≤0），且基 B 是可行的，那么基 B 就是最优的。

证明

假设存在 x∗ 满足：

{Ax∗≤b,c⊤x∗<c⊤xBABx∗≤bB

则有：

λB⊤ABx∗≥λB⊤bB

因此：

c⊤x∗≥λB⊤ABxB=c⊤xB

这与 c⊤x∗<c⊤xB 矛盾，所以不存在这样的 x∗，基 B 是最优的。

5.1 Changing basis

假设 λB 中有一些系数为正，那么我们需要沿着多面体的边移动到新的顶点（即更换基）。

示意公式：

xB′=xB+μd=AB′−1bB′

其中，xB′ 是新的基对应的顶点，μd 表示沿着多面体边的移动方向和距离。
